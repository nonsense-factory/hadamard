## Notes :

This particular Hadamard code has Reed-Miller properties of m = 4, [2^m, m+1, 2^m-1] = [16, 5, 8] 
and can correct for mfloor(2^(m-2) - 1/2)  = floor(4 - 1/2) = 3 errors per message.
